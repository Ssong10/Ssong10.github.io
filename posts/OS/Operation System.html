#OS ( 운영 체제 )

Operation System

#프로세스와 스레드

#프로그램(Program)이란

  • 어떤 작업을 위해 실행할 수 있는 파일

#프로세스(Process)란

  • 컴퓨터에서 연속적으로 실행되고 있는 컴퓨터 프로그램
  • 메모리에서 올라와 실행되고 있는 프로그램의 인스턴스(독립적인 개체)
  • 하나의 프로세스틑 최소 1개의 스레드(메인 스레드) 를 가지고 있다
  • 기본적으로 다른 프로세스의 자료에 접근할 수 없으며, 접근하려면 프로세스 간의 통신 (IPC, inter-process communication)(ex)파이프, 파일, 소켓) 등을 사용해야한다.

#스레드(Thread) 란

  • 프로세스 내에서 실행되는 여러 흐름의 단위
